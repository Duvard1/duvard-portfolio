# Duvard Portfolio

A clean, responsive personal portfolio built with HTML, CSS, and vanilla JavaScript. It highlights projects, education, and contact, with subtle motion and a mobile-first navigation experience.

## Highlights
- Responsive layout from mobile to desktop
- Animated intro marquee and scroll reveals
- Project cards with tags and clear CTAs
- Accessible nav toggle and smooth UX

## Tech Stack
- HTML5
- CSS3
- JavaScript (vanilla)

## Getting Started
1. Open `index.html` in your browser.

## Structure
- `index.html`
- `css/` for reset, variables, core styles, and responsive rules
- `js/` for animations and the mobile menu
- `assets/` for images and icons
